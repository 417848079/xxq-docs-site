# 键盘快捷键

## RK75 快捷键

| 快捷键     | 功能     |
| ---------- | -------- |
| `Fn+Enter` | 电量查询 |
| `Fn + ]`   | Insert   |

## AndroidStudio快捷键
|快捷键|功能|
|:----|:----|
|`Ctrl+Shift+F`|搜索|
|`Shift+F6`| 重命名 |

## mac访达快捷键
|快捷键|功能|
|:----|:----|
|`⌘+⇧+G`|前往文件夹|
|`⌘+⌥+C`|复制文件路径|
|`⌘+⌥+→`|下一级文件夹|
|`⌘+⌥+N`|新建文件夹|
|`⌘+⌥+C`|复制文件或文件夹|
|`⌘+⌥+V`|粘贴文件或文件夹|

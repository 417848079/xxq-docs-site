## final vs const（关键区别）

| 特性       | final                                   | const                                                      |
| ---------- | --------------------------------------- | ---------------------------------------------------------- |
| 赋值时机   | 运行时确定值                            | 编译时就确定值（写死的常量）                               |
| 修改限制   | 只能赋值一次                            | 完全不可变，连内部都不能改                                 |
| 使用场景   | 组件属性、运行时确定的不变量            | 纯常量（如固定字符串、数字）                               |
| Flutter 中 | 组件参数首选 纯静态常量、const 构造函数 | final 只能赋值一次，const 可以赋值多次，但是赋值后不可修改 |

##
